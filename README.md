# WeatherScan

WeatherScan is a web application built in Angular to view the current weather in your area.

To run this project you must have Angular and Node properly installed on your system.

## 1. Clone the Repository:

```shell
git clone https://github.com/jmfweb/WeatherScan.git
```

## 2. Navigate to the project directory:

```shell
cd your-workspace-name
cd WeatherScan
```

## 3. Install the project dependencies:

```shell
npm install
```

## 4. Run the project:

```shell
npm start
```

## 6. Access the application:
   
Once the project is running, open a web browser and visit the local address provided in the console output.
